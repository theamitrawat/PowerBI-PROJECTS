## Overview

This project presents a comprehensive Supply Chain and Logistics analytics dashboard developed using Power BI. The dashboard provides insights across key operational areas including inventory, procurement, production, sales, and shipment performance, enabling data-driven decision-making.

---

## Key Features

* Inventory monitoring and stock analysis
* Shipment and logistics performance tracking
* Procurement and supplier evaluation
* Production insights
* Sales and revenue analysis
* Interactive filters and drill-down capabilities

---

## Tools and Technologies

* Power BI – Data visualization and dashboard development
* SQL – Data extraction and querying
* Python – Data preprocessing and transformation
* CSV/Excel – Data sources
